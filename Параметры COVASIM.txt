Параметры COVASIM

----Параметры, которые можно убрать----

location - район моделирования

n_days - количество дней моделирования 

verbose - выводить или нет подробную информацию

Рескейлинг популяции:
	pop_scale 
	scaled_pop
	rescale
	rescale_threshold
	rescale_factor

dynam_layer - является ли слой динамическим

beta_layer - вирулентвность для каждого слоя

viral_dist - трансмиссивность, изменяющаяся во времени

asymp_factor - множитель для вирулентности бессимптомных

n_imports - среднесуточное количество 
беспричинных заражений

immunity - Матрица факторов иммунитета и перекрестного 
иммунитета

trans_redux - 

sev2crit - Продолжительность от тяжелого сотояния до 
критического

crit2rec - Продолжительность выздоровления людей с 
критическими симптомами

crit2die - Продолжительность от появления критических 
симптомов до смерти

rel_crit_prob - Масштабный коэффициент для доли тяжелых 
случаев, переходящих в критические

rel_death_prob - Масштабный коэффициент для доли 
критических случаев, закончившихся смертью

prog_by_age - установить ли прогрессирование болезни в 
зависимости от возраста человека

  
iso_factor - Множитель для беты для инфекционных людей в 
случае изоляции

quar_factor - Карантинный множитель по трансмиссивности и 
восприимчивости

quar_period - Количество дней карантина; предположение

interventions - Вмешательства, присутствующие в данной 
симуляции

analyzers - Пользовательские функции анализа

timelimit - ограничение времени для симуляции

stopping_func - функция для остановки симуляции во время
выполнения

n_beds_hosp - Количество больничных коек для тяжелобольных 
пациентов

use_waning - динамически рассчитываемый иммунитет или нет

n_beds_icu - Количество коек в отделении интенсивной 
терапии, доступных для тяжелобольных пациентов

no_hosp_factor - Множитель, определяющий, насколько 
возрастает вероятность того, что тяжелобольные люди 
окажутся в критическом состоянии при отсутствии мест в 
больнице

no_icu_factor - Множитель вероятности смерти тяжелобольных 
людей при отсутствии мест в отделении интенсивной терапии

vaccine_pars - Параметры вакцины

vaccine_map - виды вакцин

variants - штаммы

variant_map - виды штаммов

variant_pars - параметры штамма






----Параметры, которые можно поставить в наши значения----

pop_size - количество агентов

pop_infected - первоначально инфецированные

pop_type - тип популяции, synthpops в нашем случае

start_day - старт моделирования

end_day - конец моделирования

frac_susceptible - доля восприимчивых

contacts - количество контактов внутри слоя

beta - бета для одного контакта

n_variants - количество штаммов внутри популяции

nab_init - Параметры распределения начальных значений 
нейтрализующих антител

nab_decay - Параметры, описывающие кинетику распада 
нейтрализующих интител с течением времени

nab_boost - Мультипликативный коэффициент, применяемый 
к уровню нейтрализующих антител человека при повторном 
заражении

rel_imm_symp - Относительный иммунитет против естественной 
инфекции зависит от симптомов

exp2inf - Продолжительность от подверженности до 
заболеваемости

inf2sym - Продолжительность от инфекционного до 
симптоматического периода

sym2sev - Продолжительность от появления симптомов до 
тяжелых проявлений

asym2rec - Продолжительность выздоровления бессимптомных 
людей

mild2rec - Продолжительность выздоровления людей с 
легкими симптомами

sev2rec - Продолжительность выздоровления людей с 
тяжелыми симптомами

rel_symp_prob - Масштабный коэффициент для доли 
симптоматических случаев

rel_severe_prob - Масштабный коэффициент для доли 
симптоматических случаев, переходящих в тяжелую форму

prognoses - Фактические массивы прогнозов по возрасту
	age_cutoffs - список диапазонов возрастов
	sus_ORs - вероятность восприимчивости для возрастов
	trans_ORs - транзитивность для возрастов
	comorbidities - 
	symp_probs - вероятность появления симптомов
	severe_probs - вероятность сложного случая
	crit_probs - вероятность критического случая
	death_probs - вероятность смерти




----Параметры, которые нужно оставить в дефолтном состоянии----

rand_seed - инициализация генератора псевдослучайных чисел

beta_dist - Распределение для определения 
трансмиссивности на индивидуальном уровне

nab_kin - Создается во время инициализации симулятора 
с использованием параметров nab_decay

nab_eff - Параметры для сопоставления нейтрализующих 
антител с эффективностью

rel_beta - Относительная трансмиссивность штамма
